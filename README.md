# Antigravity Auto-Accept & Russifier

Универсальный автокликер и полный русификатор интерфейса для **Antigravity Chat 2.0** и **Antigravity IDE**.  
Автоматически нажимает кнопки Submit/Отправить, Retry/Повторить, Accept/Разрешить, а также переводит весь интерфейс настроек и разрешений на русский язык.

> Работает динамически в реальном времени — включая элементы в Shadow DOM и Iframe.

## Возможности

- 🇷🇺 **Полная русификация настроек** — переводит меню настроек безопасности, аккаунта, разрешений файлов/команд, моделей ИИ и параметров редактора на русский язык.
- ✅ **Автоклик Submit/Отправить/Send** — нажимает кнопку отправки сообщений.
- 🔄 **Автоповтор Retry/Повторить** — при ошибках сети (с настраиваемым лимитом попыток).
- 🛡️ **Accept/Разрешить** — автоматически принимает диалоги подтверждения команд и разрешений.
- 🔵 **Обход проектов** — переключается между проектами с синими индикаторами активности.
- ⏸️ **Пауза при активности** — приостанавливает клики при действиях пользователя для комфортной работы.

## Инструкция для ИИ-агентов (Cascade, Antigravity, etc.)

Если вы являетесь ИИ-агентом, выполняющим русификацию или установку автокликера:
1. Клонируйте этот репозиторий:
   ```bash
   git clone https://github.com/mdn77/antigravity-auto-accept.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd antigravity-auto-accept
   ```
3. Запустите соответствующие скрипты установки:
   - **Для Chat 2.0 (Electron-приложение):** `node install.js`
   - **Для Antigravity IDE:** `node install-ide.js`
   - **Для установки везде сразу:** `node install.js && node install-ide.js`
4. Сообщите пользователю о необходимости перезапустить приложение Antigravity.

## Установка вручную

### Требования
- Node.js 18+
- Antigravity Chat 2.0 и/или Antigravity IDE

### Быстрая установка

```bash
git clone https://github.com/mdn77/antigravity-auto-accept.git
cd antigravity-auto-accept

# Для Chat 2.0 (Electron-приложение)
node install.js

# Для Antigravity IDE
node install-ide.js

# Для обоих сразу
node install.js && node install-ide.js
```

### Удаление

```bash
# Chat 2.0
node install.js --uninstall

# IDE
node install-ide.js --uninstall
```

## Как это работает

### Chat 2.0
1. `install.js` создаёт резервную копию `app.asar` -> `app.asar.backup`.
2. Распаковывает архив и внедряет `autoclicker.js` в папку `dist/`.
3. Добавляет загрузчик в `utils.js` (через метод `executeJavaScript` при наступлении события `did-finish-load`).
4. Собирает `app.asar` обратно.

### IDE
1. `install-ide.js` создаёт резервную копию `workbench.html` -> `workbench.html.backup`.
2. Копирует `autoclicker.js` в директорию workbench как `antig_autoclicker.js`.
3. Добавляет тег `<script src="./antig_autoclicker.js" defer></script>` в конец `workbench.html`.

> ⚠️ После обновлений Antigravity может потребоваться повторный запуск установщика.

## Виджет управления

После установки и перезапуска в правом нижнем углу появится кнопка управления (**AC** или **AutoClick**):

| Настройка | Описание |
|---|---|
| **Автоклик** | Вкл/выкл автоматическое нажатие кнопок. |
| **Обход точек** | Переключение между проектами с непрочитанными уведомлениями. |
| **Повтор ошибок** | Автоматический Retry при сбоях сети/генерации. |
| **Ретраи** | Максимальное число попыток повтора (по умолчанию 10). |
| **Пауза** | Задержка автокликера после активности пользователя (в секундах). |

## Поддерживаемые кнопки автоклика

| Действие | На английском | На русском |
|---|---|---|
| Отправить | Submit, Send | Отправить |
| Повторить | Retry, Try again | Повторить, Попробовать снова |
| Разрешить | Yes allow, Always proceed, Always Allow | Да разрешить, Всегда выполнять, Разрешить всегда |
