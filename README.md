# Antigravity Auto-Accept

Универсальный автокликер для **Antigravity Chat 2.0**.  
Автоматически нажимает кнопки Submit/Отправить, Retry/Повторить, Accept/Разрешить.

> Работает на **любом языке** — до и после русификации интерфейса.

## Возможности

- ✅ **Автоклик Submit/Отправить/Send** — нажимает кнопку отправки
- 🔄 **Автоповтор Retry/Повторить** — при ошибках сети (с лимитом попыток)
- 🛡️ **Accept/Разрешить** — автоматически принимает диалоги разрешений
- 🔵 **Обход проектов** — переключается между проектами с синими индикаторами
- ⏸️ **Пауза при активности** — не кликает когда работает пользователь
- 🌐 **Универсальный** — EN + RU, не зависит от CSS-классов

## Установка (Chat 2.0)

### Требования
- Node.js 18+
- Antigravity Chat 2.0 установлен

### Быстрая установка

```bash
git clone https://github.com/mdn77/antigravity-auto-accept.git
cd antigravity-auto-accept
node install.js
```

### Удаление

```bash
node install.js --uninstall
```

Восстанавливает оригинальный `app.asar` из бэкапа.

## Как это работает

1. `install.js` создаёт бэкап `app.asar` → `app.asar.backup`
2. Распаковывает архив, внедряет `autoclicker.js` в `dist/`
3. Добавляет загрузчик в `utils.js` (метод `executeJavaScript`)
4. Собирает `app.asar` обратно

После перезапуска Antigravity виджет **AutoClick** появится в правом нижнем углу.

## Виджет управления

| Настройка | Описание |
|---|---|
| **Автоклик Submit** | Вкл/выкл автонажатие кнопок отправки |
| **Обход синих точек** | Переключение между проектами с уведомлениями |
| **Повтор при ошибках** | Автоматический Retry при сбоях |
| **Попыток Retry** | Максимум повторов (по умолчанию 10) |
| **Пауза (сек)** | Задержка после действий пользователя (по умолчанию 20с) |

## Поддерживаемые кнопки

| Действие | English | Русский |
|---|---|---|
| Отправить | Submit, Send | Отправить |
| Повторить | Retry, Try again | Повторить, Попробовать снова |
| Разрешить | Yes allow, Always proceed | Да разрешить, Всегда выполнять |
