# Antigravity Auto-Accept ⚡

> Автоматически нажимает кнопки Submit, Accept и Retry в [Antigravity IDE](https://antigravity.dev), позволяя вам сосредоточиться на работе.

![Платформа](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-blue)
![Версия](https://img.shields.io/badge/version-1.1.0-green)
![Лицензия](https://img.shields.io/badge/license-MIT-brightgreen)

## ✨ Возможности

- **🖱️ Автоклик Submit** — Автоматически нажимает «Submit», когда появляется окно с подтверждением.
- **🔵 Обход синих точек** — Сканирует панель проектов, находит активные проекты (с синим индикатором) и автоматически переключается на них для нажатия Submit.
- **🔄 Авто-повтор (Retry)** — Повторяет попытки при ошибках соединения до N раз (настраиваемо).
- **⏸️ Умная пауза** — Временно отключается, если вы сами печатаете или кликаете в окне, и возобновляет работу через заданное время.
- **🎛️ Панель управления** — Красивый виджет в углу экрана, позволяющий настраивать и отключать функции прямо во время работы.
- **📦 Простая установка** — Настройка одной командой `node install.js`.

## 📸 Интерфейс виджета

Панель настроек появляется в нижнем правом углу Antigravity:

```
┌──────────────────────────┐

├──────────────────────────┤
│ Автоклик Submit      🟢  │
│ Обход синих точек    ⚫  │
│ Повтор при ошибках   🟢  │
│ Попыток Retry:   [10]    │
│ Пауза (сек):     [20]    │
│                          │
│ Нажато: 42  Ретраи: 0/10 │
│ Активные точки: 3 Готов  │
└──────────────────────────┘
```

**Цвета индикатора работы (светодиод):**
- 🟢 Зелёный — Активен и выполняет клики.
- 🟡 Жёлтый — На паузе (пользователь работает в редакторе).
- ⚫ Серый — Выключен.

## 🚀 Установка

### Требования
- Установленный [Node.js](https://nodejs.org/) (версия 16+)
- Установленный [Antigravity IDE](https://antigravity.dev)

### Процесс установки
```bash
git clone https://github.com/mdn77/antigravity-auto-accept.git
cd antigravity-auto-accept
node install.js
```

После завершения **перезапустите Antigravity**.

### Удаление плагина
```bash
node install.js --uninstall
```
Эта команда восстановит оригинальный `app.asar` из резервной копии.

### Альтернативный запуск (через Console)
Если вы не хотите менять файлы программы, скопируйте код из файла `autoclicker.js` и вставьте его в консоль разработчика Antigravity (`Ctrl+Shift+I` → вкладка **Console**).

## 🔧 Совместимость

| Версия Antigravity | Поддержка | Примечание |
|---|---|---|
| 2.0.6+ | ✅ Полная | Инъекция через `executeJavaScript` (v1.1.0+) |
| 2.0.0–2.0.5 | ✅ Полная | Инъекция через `executeJavaScript` |

> **Примечание:** Начиная с v1.1.0 используется метод `webContents.executeJavaScript()` вместо DOM-инъекции. Это решает проблему с `contextIsolation: true` в новых версиях Antigravity.

## ⚙️ Настройки виджета

| Параметр | По умолчанию | Описание |
|---|---|---|
| **Автоклик Submit** | ✅ ON | Кликает Submit на текущей странице каждые 2 секунды. |
| **Обход синих точек**| ❌ OFF | Ищет проекты с синей точкой, заходит в них, нажимает Submit и возвращается. |
| **Повтор при ошибках**| ✅ ON | Нажимает Retry / Try Again при возникновении сетевых ошибок. |
| **Попыток Retry** | 10 | Лимит попыток ретрая до остановки (сбрасывается через 60 сек). |
| **Пауза (сек)** | 20 | Время ожидания в секундах после ваших действий на странице до возобновления работы. |

## 🛡️ Безопасность

- Перед установкой всегда создаётся **бэкап** файла `app.asar`.
- Функция **uninstall** полностью возвращает всё в исходное состояние.
- Клики по самому виджету **не сбрасывают** таймер вашей активности.
- Обход проектов **мгновенно прекращается**, если вы начинаете работать с клавиатурой или мышью.
